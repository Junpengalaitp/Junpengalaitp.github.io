Detailed Description

This class represents a trade route for an account's fcm/ib on an exchange.

The TradeRouteInfo class is used to convey the existence of an available trade route to an exchange for an fcm/ib, and to indicate its status. The trade route must be specified when placing orders.

RApi::TradeRouteInfo::iType

This member variable indicates if the trade route info is from a cache or the real-time data stream. Generally the type will be MD_IMAGE_CB when the TradeRouteInfo is the result of a call to REngine::listTradeRoutes() and will MD_UPDATE_CB when the TradeRouteInfo is the result of a call to REngine::subscribeTradeRoute().
